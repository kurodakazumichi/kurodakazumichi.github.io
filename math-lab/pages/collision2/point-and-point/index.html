点と点の衝突

点と点の衝突は非常に単純で、２つの点の座標が一致するかどうかを調べればよい。

計算方法

点Aと点Bがあり、それぞれの座標を以下の通りとした場合

  • A$(x1, y1)$
  • B$(x2, y2)$

$x1 = x2$ かつ $y1 = y2$であれば、２つの点は衝突している。

解説

解説するほどの事はないわけだが、 実際、点の座標が小数点以下含めて完全に一致するということは稀である。

さらにコンピューターには小数点誤差というものが存在するため動き回る点の座標が完全に一致するという事はほぼ起こりません。

実際、このように動きを滑らかにすると見た目上は衝突しているように見えますが 小数点誤差によってコンピューター的には当たっていないと判定されます。

サンプルコード

function intercect(p1:Vector2, p2:Vector2)
{
  return (p1.x === p2.x && p1.y === p2.y);
}
