三平方の定理

$c^2 = a^2 + b^2$

三平方の定理(ピタゴラスの定理)はベクトルの長さを求めるという用途でめちゃくちゃよく使う。

2Dベクトルは基本的に$x$, $y$成分を持っているので、三平方の定理を使えば必ず長さが求められる。

証明

$c^2$ は 外側の大きい四角から、三角形４つを引いた面積になるはずなので以下の式が成り立てばいい。

$c^2 = (a + b)^2 - 4(\frac{ab}{2})$
$ \ \ \ = a^2 + 2ab + b^2 - 2ab$
$ \ \ \ = a^2 + b^2$
